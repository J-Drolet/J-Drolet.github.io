Fall 2021 - Various Game projects using Unity

In Comp 521 at McGill (Modern Computer Games) I created several projects in Unity focused on a variety of game-related algorithms and concepts.


Projects created were:

  • Game with dynamically created trees that can be destroyed by firing projectiles. A maze is automatically generated on each run using Eller's algorithm. Play here
  • Game example to implement collision detection, dynamic terrain generation and verlet integration. The mound in generated dynamically using midpoint bisection. The insects are moved and modelled using verlet integration. Play here
  • Tech demo for effective NPC pathfinding. Both RRT (Rapidly-expanding Random Tree) and A* pathfinding algorithms were implemented. Teleporters and obstacles are created to increase the complexity of the pathfinding. Play here
  • Game to implement basic game AI. The squirrels attempt to gather acorns and then store them in their home trees. The squirrels make a plan based on the GOAP algorithm (Goal Oriented Action Planning). The squirrels are also afraid of the player and will decide to run away if the player gets close. The player can enter a "ghost mode" by presing space. Play here

Summer 2021 - Hole in the wall game using OpenGL and C++

In 2021, I took Comp 371 (Computer Graphics). Our team of 3 were tasked with using C++ and OpenGL to create a playable game in which a series of geometric shapes must be rotated to fit through a wall.


During this project, I implemented:

  • Lighting code to support point lights, spot lights, and directional lights in GLSL (OpenGL Shading Language)
  • Shaders to support shadows created by the lighting
  • Code to allow the importing of textures.
  • Gameplay logic that utilizes quarternion math in order for shape rotations to occur as expected by the player.
  • Controls to allow camera adjustments


Fall 2020 - Modeling chemical engineering problems using MATLAB

In CHEE 390 (Computational Methods in Chemical Engineering) at McGill, I implemented chemical engineering related computations using MATLAB.


Projects computed includes:

  • Implementation of the Newton-Raphson Algorithm for finding the roots of a spatial concentration equation.
  • Implementation of Gauss-Jordan elimination in order to solve a system of nonlinear equations in the context of chemical reaction equilibria.
  • Implementation of Romberg integration in the context of calculating the time spent by a rocket in the air.
  • An ODE equation solver implementing Implicit Euler, the Trapezoidal rule and Butcher's methods. This was used to calculate how the concentrations of species in a reaction changes over time using the rate constants.
